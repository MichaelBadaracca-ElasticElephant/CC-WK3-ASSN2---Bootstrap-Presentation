Bootstrap Labels & Badges

Bootstrap Labels

  • Labels can be a great way to call out keywords or mark an element on the page

  • Real World Example:

  • Code:

    <span class="label label-info">Rad Code</span>

  • Result:

    Rad Code

Bootstrap Badges

  • Badges can easily highlight new items ! to the user

  • Real World Example:

  • Code:

    <span class="badge">42</span>

  • Answer to Everything:

    42
